# Vendored package

Contents of the npm package `@postiz/wallets@0.0.1` (Apache-2.0), vendored on
2026-07-17 from the public npm registry tarball so the monorepo does not depend
on an upstream-controlled registry package. It is a re-export bundle of
`@solana/wallet-adapter-*` packages (see `repository` in package.json:
anza-xyz/wallet-adapter). Original LICENSE retained. The root workspace
consumes it via `"@postiz/wallets": "workspace:*"`, keeping the import sites
unchanged.
